### Detailed Description
```
GitHub Issue to AI Prompt - Transform GitHub Issues into Perfect AI Prompts

Tired of manually copying and formatting GitHub issues to get help from AI assistants? This extension automatically extracts all relevant information from any GitHub issue and converts it into a well-structured prompt ready for Claude, ChatGPT, or any AI coding assistant.

KEY FEATURES:

• One-Click Extraction - Automatically detects GitHub issue pages and extracts title, description, labels, code blocks, error logs, comments, and linked PRs

• AI-Powered Generation - Use OpenAI's GPT models to intelligently generate optimized prompts that capture the essence of the issue

• Multiple Templates - Choose from 6 built-in templates (Bug Fix, Feature Request, Code Review, Question/Help, Minimal, Detailed) or create your own custom templates

• Smart Detection - Extracts reproduction steps, expected vs actual behavior, environment info, and relevant technical discussions

• Visual Indicator - Green badge on the extension icon when you're on a GitHub issue page

• Auto-Generation - Optionally auto-generate prompts when you open the extension on issue pages

• Privacy First - Your API key is stored locally and only sent to OpenAI. No data collection, no tracking.

PERFECT FOR:
- Developers seeking AI help with bug fixes
- Teams using AI assistants for code reviews
- Anyone who wants faster, better context when asking AI for help

HOW TO USE:
1. Navigate to any GitHub issue
2. Click the extension icon (or enable auto-open)
3. Choose a template or click "Generate with AI"
4. Copy the optimized prompt to your clipboard
5. Paste into your favorite AI assistant

CUSTOMIZATION:
- Create unlimited custom templates
- Import/export templates as JSON
- Configure which content to include (comments, code blocks, PRs)
- Choose your preferred OpenAI model

Free to use with templates. AI generation requires your own OpenAI API key.

Open source: https://github.com/lihaoz-barry/Issue2Prompt
```
