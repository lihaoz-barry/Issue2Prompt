## Quick Canva Instructions

1. Go to https://www.canva.com
2. Create design ‚Üí Custom size (1280x800 for main, 1280x720 for YouTube)
3. Search templates for "app promo" or "software"
4. Use these elements:
   - Search "GitHub" for Octocat icon
   - Search "AI" or "sparkle" for AI icon
   - Upload your screenshots
5. Apply your colors (#238636 green, #0D1117 dark)
6. Add text with bold font
7. Download as PNG

After creating images, save as:

- `banner-twitter.png` - Twitter header

## GIF Demo Creation

For Product Hunt and social media, a GIF demo is highly effective.

### Recording
1. Use **Kap** (Mac) or **ScreenToGif** (Windows)
2. Record at 15-30 FPS
3. Keep under 5MB for most platforms

### Storyboard (10-15 seconds)
```
[0-2s]  Navigate to GitHub issue
[2-4s]  Click extension icon
[4-7s]  Show popup with generated prompt
[7-9s]  Click copy button
[9-12s] Show "Copied!" feedback
[12-15s] Optional: paste into AI chat
```

### Optimization
- Use https://ezgif.com/optimize to reduce file size
- Target: under 5MB, 480-720px width
